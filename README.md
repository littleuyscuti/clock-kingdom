# ⏰ Clock Kingdom

A bilingual (繁體中文 / English) clock-learning web game for kindergarten and primary kids.
Single HTML file — no server, no ads, no install.

## 🎮 Play now

### → [**littleuyscuti.github.io/clock-kingdom**](https://littleuyscuti.github.io/clock-kingdom/) ←

Works on any phone, tablet, or computer with a browser. No download. No account.

## What's inside

15 levels that build from "what is a clock" to reading any time + 24-hour format:

| # | Level | What you learn |
|---|-------|----------------|
| 1 | Meet the Clock | Tour of the clock |
| 2 | Hour Hand 1: Read | Hour hand points to a number |
| 3 | Hour Hand 2: O'clock | "3 o'clock" = "3:00" |
| 4 | Hour Hand 3: Set | Drag the hour hand |
| 5 | hours : mins | Digital time format |
| 6 | Minute Hand 1: every 15 min | 12/3/6/9 → 0/15/30/45 |
| 7 | Minute Hand 2: every 5 min | Number × 5 rule |
| 8 | Minute Hand 3: every 1 min | Each small tick = 1 minute |
| 9 | Hour Between | Hand between two numbers → read the smaller |
| 10 | Read Time 1: Easy | Both hands, on-the-5 minutes |
| 11 | Read Time 2: Hard | Any time, off-tick minutes |
| 12 | Set Both Hands | Drag minute first, hour follows |
| 13 | Match Clocks | Pick the clock that matches a digital time |
| 14 | 24-Hour | Morning uses 1–12, afternoon 13–24 |
| 15 | Ultimate Test | 10 questions, 90 seconds, all mixed |

## Features

- 🌏 **Bilingual** — toggle 中 / EN any time
- ⭐ **3-star scoring** per level, progress saved locally
- 🎵 Cute sound effects and friendly mascot (Ticky)
- 📱 **Phone-first** layout, works great on tablets and desktops too
- 🎯 Varied questions — no two plays feel the same

## Run locally

Just open `index.html` in any browser. Or from this folder:

```bash
python3 -m http.server 8765
# then open http://localhost:8765
```
